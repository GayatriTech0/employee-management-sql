                                       Employee Management SQL

Project Overview:

This Project is a SQL-based Employee Management System developed using MySQL Wrokbench.

It manages employee, department, and salary data using relational database concepts.


Features:

- Employee data management
- Department managment
- Salary management
- SQL JOIN operations
- GROUP BY queries and reporting
- Relational database structure using foreign keys

Technologies Used:

- MySQL
- MySQL Workbench
- SQL

Database Table:

- employee_data
- departments
- salaries


Operations Performed:

- creating tables
- Importing CSV datasets
- Using PRIMARY KEY and FOREIGN KEY
- Performing JOIN queries
- Generating employee salary reports

Data Visualization:

- Created salary distribution visualizations using Tableau Public
- Visualized department-wise employee count using bar charts
- Used charts to support SQL-based analysis and reporting
